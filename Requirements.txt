How to Run

1. Clone the Repository

git clone https://github.com/your-username/61hrt.git
cd 61hrt

2. Set Up the Environment
Install the dependencies:

pip install streamlit requests

3. Get a Perplexity API Key
You will need a free or paid API key from Perplexity Labs.

Due to security reasons, the API key is not included in this repository.
Once you get your API key, paste it inside the file where it says:

API_KEY = "your-perplexity-api-key-here"

4. Run the App
Use the following command to launch the app locally:

python -m streamlit run LLM.py
